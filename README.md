# 🎬 AutoSubtitle-DualAgent: 本地化双 Agent 智能字幕流水线

这是一个基于 Vibe Coding 理念快速构建的本地化字幕生成与翻译工具。主打**纯本地运行、零隐私泄露、全自动流转**。

利用 `faster-whisper` 进行高精度原声音频提取，配合本地部署的 LLM（推荐 Qwen 等强力模型，通过 LM Studio 驱动）实现“双 Agent 流水线”：**Agent 1 负责根据语境修复语音识别错别字，Agent 2 负责高质量跨语言翻译**。

## ✨ 核心特性

- 🔒 **100% 本地化处理：** 断网可用，无云端 API 依赖，彻底告别隐私焦虑。
- 🧠 **双引擎断句路由：** 针对中日文（CJK）采用原生段落引擎，针对英语等表音文字采用前瞻容忍吸附引擎，避免字幕碎裂或溢出。
- 🤖 **双 Agent 协作：** - 校对 Agent：自动修复同音字、标点遗漏等 ASR 幻觉。
  - 翻译 Agent：结合上下文语境进行高质量中文化翻译。
- 🧹 **内存自动回收：** 阶段任务完成后，自动向 LM Studio 发送指令卸载模型，交还操作系统显存。
- 🎯 **一键出件：** 支持拖拽单文件或文件夹，全自动输出带双语/单语样式的 `.ass` 及 `.srt` 文件。

## ⚠️ 必看：运行前必读（硬编码提示）

本项目保留了强烈的“自用开发”痕迹，为保证脚本正常运行，你需要注意以下几点环境配置：

### 1. 必须安装 FFmpeg（核心依赖项！）
脚本在处理视频提取音频时高度依赖 FFmpeg。如果缺失，程序将跳过处理或报错。
* **Windows 用户：** 请前往 [FFmpeg 官网](https://ffmpeg.org/download.html) 下载，解压后**必须将其 `bin` 目录添加到系统的环境变量 `Path` 中**。
* 安装完成后，在终端输入 `ffmpeg -version` 确认可用。

### 2. 需要修改源码的本地路径与配置
请打开 `auto_subtitle - V32 细节优化.py`，根据你的本地环境修改以下变量：
* **`WHISPER_DOWNLOAD_ROOT`** (第 28 行): 默认写死了 `r"G:\WhisperModels"`，请改为你本地存放 Whisper 模型的路径。
* **`LLM_BASE_URL`** (第 20 行): 默认指向 LM Studio 的本地 API `http://127.0.0.1:1234/v1`，如有变更请修改。
* **`MODEL_NAME`** (第 24 行): 默认为 `"Qwen3.5-27B"`，**请务必确保该名称与你 LM Studio 内加载的模型 ID 完全一致**，否则 Agent 无法调用。

## 🚀 快速起步

### 环境准备
1. 安装 Python 3.10+
2. 克隆本仓库并安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

### 运行依赖
1. 启动 **LM Studio**，加载你喜欢的模型（推荐 Qwen 系列），并开启 Local Server 监听 1234 端口。
2. 确保系统已配置好 FFmpeg。

### 开始使用
直接运行主脚本：
```bash
python "auto_subtitle - V32 细节优化.py"
```
根据终端提示：
1. 输入执行模式（1: 听译全流 / 2: 仅听写 / 3: 仅对现有 SRT 翻译）。
2. 将需要处理的音视频文件（支持 mp4, mkv, mp3, wav 等）或所在文件夹直接**拖入终端窗口**并回车。
3. 喝杯咖啡，等待双语字幕生成。
